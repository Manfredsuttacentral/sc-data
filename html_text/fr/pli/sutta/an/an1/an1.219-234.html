• Aṅguttara Nikāya

Les disciples les plus distingués (4)

1.219

Bhikkhous, le plus distingué parmi mes disciples bhikkhous très instruits est Ananda.

1.220

Bhikkhous, le plus distingué parmi mes disciples bhikkhous ayant une bonne mémoire est Ananda.

1.221

Bhikkhous, le plus distingué parmi mes disciples bhikkhous apprenant rapidement est Ananda.

1.222

Bhikkhous, le plus distingué parmi mes disciples bhikkhous résolus [à apprendre par cœur] est Ananda.

1.223

Bhikkhous, le plus distingué parmi mes disciples bhikkhous ayant été assistants personnels est Ananda.

1.224

Bhikkhous, le plus distingué parmi mes disciples bhikkhous ayant une grande assemblée [de dévôts] est Ourouvélakassapa.

1.225

Bhikkhous, le plus distingué parmi mes disciples bhikkhous inspirant la confiance aux familles est Kaloudayi.

1.226

Bhikkhous, le plus distingué parmi mes disciples bhikkhous en bonne santé est Bakoula (MN 124).

1.227

Bhikkhous, le plus distingué parmi mes disciples bhikkhous se remémorant leurs existences passées est Sobhita.

1.228

Bhikkhous, le plus distingué parmi mes disciples bhikkhous connaissant par cœur la Discipline est Oupali.

1.229

Bhikkhous, le plus distingué parmi mes disciples bhikkhous exhortant les bhikkhounis est Nandaka (MN 146).

1.230

Bhikkhous, le plus distingué parmi mes disciples bhikkhous vigilants à l'entrée des facultés sensorielles est Nanda (AN 8.9).

1.231

Bhikkhous, le plus distingué parmi mes disciples bhikkhous exhortant les bhikkhous est Mahakappina (SN 21.11, SN 54.7).

1.232

Bhikkhous, le plus distingué parmi mes disciples bhikkhous habiles avec l'élément feu est Sagata (Pc 51).

1.233

Bhikkhous, le plus distingué parmi mes disciples bhikkhous recevant des discours éloquents est Radha (SN 22.71, SN 23, SN 35.76 à 78).
