• Aṅguttara Nikāya

Répétitions sur une acquisition

1.378

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui séjourne en forêt.

1.379

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui se nourrit d'aumônes.

1.380

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui porte des haillons abandonnés comme déchets.

1.381

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui porte seulement trois robes.

1.382

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui fait des discours sur le Dhamma.

1.383

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un qui connaît la Discipline par cœur.

1.384

Véritablement, bhikkhous, ceci est une acquisition: être quelqu'un de très instruit.

1.385

Véritablement, bhikkhous, ceci est une acquisition: être un ancien.

1.386

Véritablement, bhikkhous, ceci est une acquisition: avoir bonne allure.

1.387

Non traduit.

1.388

Non traduit.

1.389

Véritablement, bhikkhous, ceci est une acquisition: être d'une bonne famille.

1.390

Véritablement, bhikkhous, ceci est une acquisition: avoir belle allure.

1.391

Véritablement, bhikkhous, ceci est une acquisition: être un bon orateur.

1.392

Véritablement, bhikkhous, ceci est une acquisition: avoir peu de désirs.

1.393

Véritablement, bhikkhous, ceci est une acquisition: être en bonne santé.
